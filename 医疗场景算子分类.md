# 医疗场景算子分类表

> 本文档依据医疗应用场景对算子进行一二级分类：**一级分类**按医疗场景划分，**二级分类**按算子能力划分（隐私脱敏、数据加密、质量过滤、质量检测、格式转换、内容清洗、内容增强、内容变换、内容识别、内容分析、内容提取、内容理解、内容安全检测、内容安全过滤、数据管理），并对算子描述进行了面向医疗场景的改写。

---

## 算子类型

> 算子按所处理的数据模态分为五种类型：**文本**、**图片**、**视频**、**音频**及**通用**。每种类型专注于对应模态的医疗数据处理，通用类算子适用于任意模态的文件管理操作。

| 算子类型 | 说明 | 算子数量 |
|---|---|---|
| 🎵 音频 | 处理诊疗录音、医疗音频等音频模态数据，包括格式转换、降噪、质量过滤、语音识别、敏感信息过滤等 | 21 |
| 📄 文本 | 处理患者病历、诊断报告、医疗文档等文本模态数据，包括内容清洗、隐私脱敏、格式转换、毒性检测、语义分析等 | 38 |
| 🖼️ 图片 | 处理医学影像、患者照片等图片模态数据，包括质量过滤、格式转换、图像增强、内容识别、目标检测等 | 40 |
| 🎬 视频 | 处理手术录像、诊疗视频等视频模态数据，包括质量检测、格式转换、视频增强、关键帧提取、内容分析等 | 22 |
| ⚙️ 通用 | 适用于任意模态文件的数据管理操作，包括文件透传、文件加密/解密、文件去重等 | 4 |
| **合计** | | **125** |
